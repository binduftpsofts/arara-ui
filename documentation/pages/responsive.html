Breakpoints

ArarA is mobile-first by default. Every utility class can be applied at specific breakpoints using modifier prefixes, allowing you to build adaptive interfaces without writing custom media queries.

The Modifier System

Apply a utility only when the screen width is equal to or greater than a specific breakpoint by prefixing the class with the breakpoint name followed by a colon.

<!-- Blue on mobile, Red on medium screens, Green on large screens -->
<div class="bg-blue-500 md:bg-red-500 lg:bg-green-500">…</div>
