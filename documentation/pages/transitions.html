Transitions & Animation

ArarA provides a smooth transition system to make your interface feel alive. Control durations, easings, and interactive transforms with simple utility classes.

Transition Property

Control which properties should transition.

HTML
<button class="transition-all duration-300 hover:bg-indigo-600 ...">
  transition-all
</button>

Transforms

Combine scale, rotate, and translate utilities smoothly.

scale + rotate

translate-x-full

HTML
<div class="transition-transform hover:scale-125 hover:rotate-12"></div>
<div class="transition-transform hover:translate-x-full"></div>
